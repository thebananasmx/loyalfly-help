# Flujos de Usuario en Loyalfly

Conoce cómo interactúan tus clientes y tu equipo con el programa de lealtad digital de Loyalfly. Hemos diseñado estos flujos para que sean rápidos, intuitivos y sin fricciones.

## 1. Flujo del Cliente: Obtener la Tarjeta

El cliente no necesita descargar ninguna aplicación adicional. Todo funciona a través de las herramientas nativas de su teléfono.

1. **Escaneo del QR:** El cliente escanea el código QR de tu negocio (ubicado en la mesa, mostrador o ticket).
2. **Registro Rápido:** Se abre una página web donde el cliente ingresa su nombre y correo electrónico (o teléfono).
3. **Guardar en Wallet:** El cliente toca el botón "Añadir a Apple Wallet" o "Añadir a Google Wallet".
4. **¡Listo!** La tarjeta digital ya está en su teléfono, lista para ser usada.

## 2. Flujo del Cliente: Acumular Sellos

Dar un sello es un proceso de 3 segundos en la caja.

1. **Mostrar Tarjeta:** Al momento de pagar, el cliente abre su Apple Wallet o Google Wallet y muestra su tarjeta de Loyalfly.
2. **Escaneo del Negocio:** El cajero o mesero escanea el código QR que aparece en la tarjeta del cliente usando la app de escáner de Loyalfly (o la cámara de un dispositivo autorizado).
3. **Sello Otorgado:** El sistema registra la visita/compra y la tarjeta del cliente se actualiza automáticamente mostrando un nuevo sello.
4. **Notificación:** El cliente recibe una notificación push confirmando su nuevo sello.

## 3. Flujo del Cliente: Canjear Recompensa

Cuando el cliente alcanza la meta (ej. 5 sellos), el proceso de canje es transparente.

1. **Notificación de Meta:** El cliente recibe una notificación de que ha ganado una recompensa. La tarjeta cambia su diseño para indicar que hay un premio disponible.
2. **Reclamar Premio:** El cliente muestra la tarjeta en el negocio.
3. **Validación:** El cajero escanea la tarjeta. El sistema indica que el cliente tiene una recompensa disponible y pregunta si desea canjearla.
4. **Confirmación:** El cajero confirma el canje. La tarjeta del cliente se reinicia a cero sellos (o se restan los puntos correspondientes) y se entrega el premio.

## 4. Flujo del Negocio: Creación del Programa

Configurar tu programa de lealtad toma menos de 5 minutos.

1. **Registro:** Creas tu cuenta de negocio en Loyalfly.
2. **Diseño de Tarjeta:** Subes tu logo, eliges tus colores corporativos y seleccionas el tipo de programa (sellos o puntos).
3. **Definir Recompensa:** Estableces cuántos sellos se necesitan y cuál será el premio (ej. "1 Café Gratis").
4. **Generar QR:** El sistema genera tu código QR único.
5. **Imprimir y Exhibir:** Descargas el material promocional (tent cards, stickers) y lo colocas en tu local.
